# Undercut AI 🏎️🏁

AI-powered Formula 1 race engineering assistant that analyzes telemetry, timing, tire strategy, and race data to generate engineer-style debriefs and strategic insights.

## Goals

- Analyze race strategy
- Compare driver pace
- Track tire degradation
- Generate engineer-style race reports
- Build reusable Claude Skills for F1 analysis

## Tech Stack

- Python
- FastF1
- Claude Skills
- Pandas
- Matplotlib
